A clean, modern **Gomoku (Five in a Row)** game for iOS with **online multiplayer** and **party mode** (local multiplayer on one device).

## Features

- **Online Multiplayer**: play against real players
- **Party Mode**: two players on the same device
- Classic Gomoku rules (place **five stones in a row** to win)
- Smooth, responsive gameplay
- Minimal UI focused on strategy
